# Navigate From Attendance Request Back To The Origin Meeting Page

## Overview
This guide provides simple instructions on how to return to your original meeting page after accessing the attendance request page. Following these steps will help you navigate smoothly between pages related to your meetings, ensuring that you can manage your schedule efficiently.

## Step-by-Step Guide

1. **Open Your Meeting Page**
   - Start by accessing your meeting page using the relevant meeting ID. This will take you to the primary information about your meeting.
   
   ![Image](../media\meetings\navigate_from_attendance_request_back_to_the_origin_meeting_page\Navigate_from_attendance_request_back_to_the_origin_meeting_page_1.png)

2. **Navigate to Attendance Request**
   - Look for the option to request attendance. Click on it to move to the attendance request page for your meeting.

   ![Image](../media\meetings\navigate_from_attendance_request_back_to_the_origin_meeting_page\Navigate_from_attendance_request_back_to_the_origin_meeting_page_2.png)

3. **Wait for Attendance Page to Load**
   - Ensure that the attendance request page has fully loaded. This will ensure that all features are available for you to review.

4. **Open Attendance Request Page**
   - If necessary, open the attendance request page again by entering the meeting ID.

5. **Go Back to Meeting Page**
   - Locate and click on the 'Go Back' option. This will return you to the original meeting page you accessed initially.

   

6. **Ensure Meeting Page is Loaded**
   - Confirm that the original meeting page is displayed correctly before proceeding with any other actions.

7. **Close the Request Attendance Page**
   - Once you have returned to the meeting page, you can close the attendance request page if you no longer need it.
